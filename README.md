# Инструкция

1. Склонируйте репозиторий:

```bash
   git clone https://github.com/Natrix111/lab-3.git
```

2. Перейдите в директорию проекта:

```bash
    cd microblog
```

3. Установите необходимые библиотеки:

```bash
    pip install django pillow
```

4. Запустите сервер:

```bash
    python manage.py runserver
```
